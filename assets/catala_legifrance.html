LEGIFRANCE_CATALA

NAME

legifrance_catala - LegiFrance interaction tool for Catala

SYNOPSIS

legifrance_catala [OPTION]â¦ FILE CLIENT_ID CLIENT_SECRET

ARGUMENTS

CLIENT_ID (required)

LegiFrance PISTE API Oauth client id

CLIENT_SECRET (required)

LegiFrance PISTE API Oauth client secret

FILE (required)

Name of the Catala master file you want to get LegiFrance information on

OPTIONS

-c VAL, --custom_date=VAL

Use in combination with -e. Instead of checking whether articles are expired now, check their expiration with respect to a custom date in the format YYYY-MM-DD.

-d, --debug

Prints debug information

-D, --diff_check

Checks the text of the articles with a LÃ©giFrance ID tag to see if there are differences with the official record.

-e, --expiration_check

Checks the expiration dates of articles with a LÃ©giFrance ID tag to see if they have expired as of now.

COMMON OPTIONS

--help[=FMT] (default=auto)

Show this help in format FMT. The value FMT must be one of auto, pager, groff or plain. With auto, the format is pager or plain whenever the TERM env var is dumb or undefined.

--version

Show version information.

EXIT STATUS

legifrance_catala exits with the following status:

0

on success.

1

on error

123

on indiscriminate errors reported on standard error.

124

on command line parsing errors.

125

on unexpected internal errors (bugs).
